Configuração Inicial

Configure as credenciais para conectar ao Strava e publicar no GitHub.

Acesso
GitHub

Gere em GitHub → Settings → Developer settings → Personal access tokens. Permissão necessária: repo (Contents: write).

Strava

Em strava.com/settings/api → crie ou acesse seu app.

Para obter: autorize seu app Strava com scope activity:read_all e copie o refresh_token retornado.
